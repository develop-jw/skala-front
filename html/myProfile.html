동양배 최적 생육 온도 예측

동양배 꽃 만개 후 일수에 따른 과당 함량이 최대가 되는 최고기온의 조건을 분석한 연구입니다.

GRU-Autoencoder DBSCAN SVR

기간: 2024.05 ~ 2024.12 (8개월)
팀 구성: 3인 (석사 연구원 1명, 대학생 1명, 본인)
담당 역할: 데이터 전처리, 모델 설계, 분석

GRU-Autoencoder 기반 특성 추출 → DBSCAN 기반 당 함량 최대 시점 필터링 → SVR 기반 예측 순서로 진행했습니다. MSE 1.08, R² 0.81의 결과를 도출했으며, 데이터 수 제한이라는 현실적 제약을 인식하고 추가 데이터 확보의 필요성을 도출했습니다.

뇌 MRI 종양 분할 성능 개선 프로젝트

의료 영상 데이터를 활용해 뇌 MRI 이미지의 종양 분할 성능을 개선하는 캡스톤 연구입니다.

Python PyTorch U-Net

기간: 2025.09 ~ 2025.12 (4개월)
팀 구성: 3인 팀 프로젝트 (창의융합DS 캡스톤)
담당 역할: 데이터 전처리, 소형 종양 탐지 성능 분석

데이터 속성 기반 층화 분할 → 결측 이미지에 대한 추가 채널 구축 → ResU-Net 모델 기반 종양 분할 순서로 진행했습니다. IoU 0.89, Dice 0.925의 결과를 도출했으며, 추가 성능 검증을 위해 테스트 데이터 셋에 대한 F1 Score 95.24%를 확인했습니다.

Python Streamlit

기간: 2026.04.30 ~ 2026.05.14
팀 구성: 3인 (대학생 2명, 본인)
담당 역할: 데이터 분할 방법 정의, 대시보드 설계
